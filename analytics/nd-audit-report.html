Vấn đề & Giải pháp

1
Script lạ từ IP domain — Rủi ro bảo mật
🔴 Critical
▾
Script sdk.min.js từ xow8cscockos4cc8oggg8co4.35.198.216.143.sslip.io load trên mọi trang. Domain dạng IP-via-sslip.io không phải pattern vendor hợp lệ đã biết. Có thể là plugin bị compromise hoặc tracking vendor kém chất lượng.
🅰️ REBUILD
  • Audit toàn bộ plugin/theme khi rebuild — không migrate script không rõ nguồn
  • Chỉ whitelist các vendor đã biết trong CSP header
🅱️ OPTIMIZE
  • Cài Query Monitor plugin → tìm script origin
  • WP Admin → Plugins → tìm plugin inject script sslip.io
  • Disable plugin ngay cho đến khi xác minh an toàn
  • Effort: 1-2 giờ | Impact: loại bỏ rủi ro bảo mật + tăng tốc độ
2
Meta description /dich-vu/ = "Khác" — Placeholder bị sót
🔴 Critical
▾
Trang /dich-vu/ — trang quan trọng nhất cho service overview — có meta description là từ "Khác". Khi Google hiển thị snippet này, CTR sẽ rất thấp. Đây là placeholder Rank Math bị sót khi config.
🅰️ REBUILD
  • Setup content template với validated meta descriptions. CMS enforce required fields trước khi publish.
🅱️ OPTIMIZE
  • Rank Math → Edit /dich-vu/ → điền meta description 140-155 ký tự, có keyword + CTA
  • Ví dụ: "Khám phá 40+ dịch vụ thẩm mỹ tại TMV Ngọc Dung: phun xăm, triệt lông, trị nám, chăm sóc da. 28 năm kinh nghiệm, 16 chi nhánh toàn quốc."
  • Kiểm tra toàn bộ service sub-pages cùng lúc
  • Effort: 1 giờ | Impact: tăng CTR từ SERP
3
sitemap_index.xml trả về 404
🔴 Critical
▾
robots.txt khai báo sitemap tại /sitemap_index.xml nhưng URL trả về 404. Google Search Console không biết cấu trúc site, ảnh hưởng indexing 100+ trang service và chi nhánh.
🅰️ REBUILD
  • Implement sitemap tự động với Next.js sitemap module hoặc equivalent
🅱️ OPTIMIZE
  • WP Admin → Settings → Permalinks → Save Changes (flush)
  • Verify sitemap URL trong Rank Math → Sitemap
  • Re-submit sitemap trong Google Search Console
  • Effort: 30 phút | Impact: Google crawl đầy đủ site
